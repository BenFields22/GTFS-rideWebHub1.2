  • Welcome to the GTFS-ride WebHub. This page provides you with a suite of applications to support the GTFS-ride data standard. Information regarding the standard can be found at https://github.com/ODOT-PTS/GTFS-ride. This tool suite provides the ability to perform the following tasks:
    - Validate GTFS-ride feeds
    - Enter GTFS-ride feeds into a central data repository
    - Create new GTFS-ride feeds
    - Run analytics on ridership data contained within the data repository

  • A fileset intended to represent GTFS-ride feed should be validated using the GTFS-ride validation tool before uploading to ensure compatibility with the central data repository and the report generation tools. To validate a GTFS-ride feed please use the GTFS-ride feed validation tool found at https://github.com/ODOT-PTS/transitfeed-ride.
